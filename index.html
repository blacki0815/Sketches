Firebase einrichten (kostenlos):
1. console.firebase.google.com → Neues Projekt erstellen
2. Realtime Database → Erstellen → Testmodus wählen
3. Projekteinstellungen → Web-App hinzufügen → Config kopieren
4. Die databaseURL sieht aus wie: https://xxx.firebaseio.com
